5. Short Long Short (String)

Link Codewars

Dadas 2 strings, a e b, retorne uma string no formato menor+maior+menor, com a string menor por fora e a string maior por dentro. As strings não serão de tamanho igual, mas elas podem ser vazias ( length 0 )

Por exemplo:

solution('1', '22') # retorna '1221'

solution('22', '1') # retorna '1221'
