Estruturas de decisão

Temos como algumas estruturas de decisão como if, elif e else.

if -> executado apenas quando a condição for verdadeira.

if 1 == 1:
    print('passsou no if')

> passou no if

if... elif -> executa apenas a condição que for verdadeira.

if 1 == 2:
    print('passsou no if')
    
elif 1 == 1:
    print('passou no elif')

> passou no elif

if... else -> executa o if se a condição for verdadeira, senão passa para o else.

if 1 == 2:
    print('passsou no if')
    
else:
    print('passou no else')

> passou no else
