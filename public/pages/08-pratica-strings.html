1. Reversed Words

Complete a solução de modo que ela inverta todas as palavras da string passada por parâmetro

Exemplo:

reverseWords('A maior vitória é a que não requer nenhuma batalha')
# Deverá retornar:
'batalha nenhuma requer não que a é vitória maior A'

Para realizar essa prática utilize este Link do Codewars.

 

2. Short Long Short

Dadas 2 strings, a e b, retorne uma string no formato menor+maior+menor, com a string menor por fora e a string maior por dentro. As strings passadas nunca serão de tamanho igual, mas elas podem ser vazias ( length 0 )

Por exemplo:

solution('1', '22') # retorna '1221'

solution('22', '1') # retorna '1221'

Para realizar essa prática utilize este Link do Codewars.

3. Returning Strings

Faça uma função que irá retornar uma mensagem de boas vindas que utilize uma entrada name. Seu programa deverá retornar:

greet('Luan')
> 'Hello, Luan how are you doing today?'

 

[Tenha certeza de escrever exatamente como o exemplo ou o programa pode não ser executado corretamente]
