Vídeo Explicativo

Entrega

Olá, devs. Nessa prática, o objetivo é criar uma agenda telefônica para encontrar e registrar facilmente os nossos contatos.

  • Crie um arquivo chamado main.py
  • Implemente todas as funções de acordo com as descrições.

Exercícios

     1 -> Crie uma lista, chamada lista_de_contatos, ela deve estar inicialmente vazia.

     2 -> Crie uma função chamada registrar_contato, ela deve ter a seguinte estrutura:

  • registrar_contato

    • Parâmetros:
      • lista: deve receber a lista_de_contatos que estará inicialmente vazia.
      • nome: deve receber o nome da pessoa, deve ser uma string. 
      • telefone: Deve registrar o telefone da pessoa, deve ser uma string com o formato livre (vai que você quer guardar um contato de outro país).
      • cidade : Deve ser uma string com o nome da cidade em que a pessoa mora.
      • profissao : Deve receber a profissão da pessoa, caso não saiba, deve ser uma string vazia.
    • Procedimento:
      • Deve atualizar a lista de contatos passada por parâmetro com um dicionário contendo todas as informações também passadas por parâmetro.
    • Retorno: Deve retornar a lista de dicionários com as informações atualizadas.
    • Importante: Os telefones não podem se repetir. 
      # EXEMPLO 1:

contato_registrado = registrar_contato(lista_de_contatos, "Ted Evelyn Mosby", "11 98765-4321", "Nova Iorque", "Arquiteto")

print(contato_registrado)

> [{
        "nome": "Ted Evelyn Mosby",
        "telefone": "11 98765-4321",
        "cidade": "Nova Iorque",
        "profissao": "Arquiteto"
   }]

 

     3 -> Crie uma segunda função chamada buscar_contato, que deve seguir a seguinte estrutura:

  • buscar_contato

    • Parâmetros:
      • lista: Deve receber a lista_de_contatos.
      • nome: Que deverá ser o nome do contato que a pessoa está procurando, no formato string.
    • Procedimento: Deve filtrar uma lista de dicionários de todos os usuários, cuja as três primeiras letras comecem com as mesmas 3 primeiras letras dos nomes passados por parâmetro.
    • Retorno: Deve a lista de dicionários com todos os usuários filtrados.
      # EXEMPLO 2: 

contatos_filtrados = buscar_contato(lista_de_contatos, "Marshall")

print(contatos_filtrados)

  • alterar_contato
    • Parâmetros:
      • lista: Deve receber a lista_de_contatos.
      • dic: Um dicionário com os atributos que deseja alterar.
      • nome: Nome do usuário que deseja realizar as alterações, em formato string.
    • Procedimento:
      • Deve atualizar o dicionário com os dados passados por parâmetro.
    • Retorno: Deve retornar uma lista de dicionários com os dados do novo contato atualizado.

      # EXEMPLO 3

contato_atualizado = alterar_contato(lista_de_contatos, {"telefone": "11 78910-1098", "profissao": "Engenheiro"}  ,"Ted Evelyn Mosby")
